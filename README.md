# DevSecOps Project

A complete cybersecurity project covering DevSecOps, Docker Security, and SSDLC.

## Labs Covered

### Lab 1 - DevSecOps CI/CD Pipeline
- Jenkins pipeline with automated security scanning
- SonarQube for Static Code Analysis (SAST)
- Trivy for container vulnerability scanning
- OWASP ZAP for Dynamic Application Testing (DAST)

### Lab 2 - SSDLC Vulnerable PHP Application
- Deliberately vulnerable PHP web app
- SQL Injection, XSS, CSRF, RCE vulnerabilities
- Complete secure fixed version included

### Lab 3 - Docker Security
- Insecure vs Secure Dockerfile comparison
- Non-root container implementation
- CVE scanning with Trivy
- Runtime security controls

## Tools Used
Docker · Git · Jenkins · SonarQube · OWASP ZAP · Trivy · PHP · MySQL
